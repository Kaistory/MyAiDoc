# Chào mừng đến với My AI Doc

Đây là không gian cá nhân nơi mình lưu trữ tài liệu, ghi chú, và những gì học được trong quá trình tự học. Site được xây dựng bằng [Docusaurus](https://docusaurus.io/) và host miễn phí trên GitHub Pages.

## Cấu trúc

Tài liệu được tổ chức theo chủ đề:

- **Lập trình** — note về ngôn ngữ, framework, kỹ thuật code
- **AI & Machine Learning** — khái niệm, mô hình, paper
- **Công cụ** — CLI, editor, workflow
- **Ghi chú chung** — những điều linh tinh khác

## Cách thêm tài liệu mới

Mỗi tài liệu là một file `.md` hoặc `.mdx` trong thư mục `docs/`. Cấu trúc thư mục chính là cấu trúc sidebar của site.

Ví dụ: tạo file `docs/lap-trinh/javascript-closure.md` với nội dung:

```markdown
---
title: JavaScript Closure
---

# JavaScript Closure

Closure là...
```

Sau đó `git push` lên GitHub, GitHub Actions sẽ tự build và deploy.

## Lệnh thường dùng

```bash
npm start          # chạy dev server (http://localhost:3000)
npm run build      # build static cho production
npm run serve      # preview bản đã build
```
