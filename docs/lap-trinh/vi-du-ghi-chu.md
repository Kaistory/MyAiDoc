# Ví dụ — ghi chú đầu tiên

Đây là file mẫu để minh họa cách viết ghi chú. Xóa hoặc thay thế bằng nội dung của bạn.

## Hỗ trợ Markdown đầy đủ

- **In đậm**, *in nghiêng*, `inline code`
- Danh sách
- Bảng biểu
- Link [Docusaurus docs](https://docusaurus.io/docs)

## Code block có syntax highlighting

```js
function hello(name) {
  return `Xin chào, ${name}!`;
}
```

```python
def hello(name: str) -> str:
    return f"Xin chào, {name}!"
```
