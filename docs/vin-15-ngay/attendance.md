# Attendance — Lộ trình Vin 15 ngày

Theo dõi tiến độ học tập qua 15 ngày, từ nền tảng AI/LLM đến triển khai Agent thực tế trong production.

> Đánh dấu ✅ khi hoàn thành xong mỗi ngày.

## Mục tiêu chương trình

Sau 15 ngày, bạn có thể:

1. **Hiểu nền tảng**: cách LLM hoạt động, kiến trúc Transformer, token, embedding.
2. **Xây Agent**: ReAct, Tool Calling, Multi-Agent, MCP/A2A.
3. **Triển khai RAG**: vector store, hybrid search, re-ranking.
4. **LLMOps**: monitoring, evaluation, guardrails, HITL.
5. **Production-ready**: deploy lên cloud, theo dõi chi phí, scale.

## Cách học hiệu quả

:::tip Gợi ý
- Mỗi ngày dành **2–3 giờ** đọc + làm bài tập.
- Code lại ví dụ, đừng chỉ đọc.
- Có blocker thì ghi lại, tra cứu/hỏi sau.
- Cuối ngày tự viết **3 câu** tóm tắt vào blog học tập.
:::
