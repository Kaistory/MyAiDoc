🔧 Roadmap Firmware Engineer (Middle Level)

Vị trí mục tiêu: Firmware Engineer 3+ năm · Công ty: Dracaena · Lương: 25 – 40 triệu/tháng · Địa điểm: Sao Mai Building, Hà Nội

🚀 Dự án từ d2l.ai phù hợp với vị trí Firmware/Edge AI

D1. LeNet trong C thuần cho MCU

Học Ch.5 + Ch.7, train LeNet trên Fashion-MNIST bằng PyTorch, sau đó port forward pass sang C99 thuần (không dùng framework), chạy trên STM32F4 hoặc Raspberry Pi Pico.

PyTorch → C99 → STM32, fixed-point math

D2. ResNet INT8 trên ARM

Train ResNet-18 trên CIFAR-10 (Ch.8), quantize INT8 bằng PyTorch quantization, deploy lên Orange Pi 5 / Rockchip RK3588 NPU bằng RKNN-Toolkit.

PyTorch quant, ONNX, RKNN, ARM NEON

D3. Transformer mini scratch

Implement Transformer encoder-decoder cho machine translation (Ch.11) từ đầu bằng NumPy, sau đó tối ưu bằng C++ với Eigen. Compare hiệu năng với PyTorch.

NumPy → C++ Eigen, profiling

D4. Edge LLM benchmark suite

Chạy Qwen2.5-0.5B/Phi-3-mini/Llama-3.2-1B trên Pi 5, Orange Pi 5, Jetson Nano bằng llama.cpp. Đo token/s, RAM, watt. Áp dụng kiến thức Ch.13.

llama.cpp, GGUF, ARM NEON, Vulkan

D5. RL cho DVFS Miner

Áp dụng Q-Learning (Ch.17) để học chính sách điều khiển voltage/frequency của hashboard theo nhiệt độ → tối ưu hashrate/watt. Triển khai trên firmware mining test.

Python (train) → C (deploy), Q-table on flash

D6. YOLO-nano edge inference

Train detection model nhỏ (Ch.14), export ONNX, deploy lên USB camera + Orange Pi. Realtime ≥ 15 FPS. Có MQTT publish lên AWS IoT.

YOLOv5n, ONNX Runtime, OpenCV, MQTT

D7. BERT-tiny sentiment classifier

Fine-tune DistilBERT/TinyBERT (Ch.15-16), quantize, chạy trên CPU ARM. Pipeline: nhận message qua MQTT → classify → publish kết quả.

HuggingFace, ONNX, ARM

D8. Custom autograd engine

Tự viết engine autograd C++ ~300 dòng (theo style micrograd của Karpathy), train MLP solve XOR. Hiểu sâu Ch.2 + Ch.5.

C++17, template, computational graph

D9. Hyperparameter tuning farm

Áp dụng Ch.19 — chạy ASHA scheduler trên cluster mining rig idle để tune hyperparam model edge. Kết hợp AWS Spot.

Ray Tune, AWS, Python

D10. OTA model update

Pipeline: train model trên cloud (Ch.23) → upload S3 → device pull qua MQTT → swap model atomically (A/B model partition).

SageMaker, S3, AWS IoT Jobs, C++

🚀 Dự án portfolio nên làm (quan trọng nhất)

Một dự án thực thay được 1 năm kinh nghiệm trên CV. Làm 3-4 cái dưới đây là đủ apply middle.

1. Mini-miner Controller

Raspberry Pi điều khiển 1 board ASIC/CPU miner qua UART, implement Stratum V1 client, gửi share lên pool thật, log lên AWS IoT.

C, pthread, libcurl, OpenSSL, JSON-RPC, MQTT

2. Edge LLM Device

Chạy Qwen2.5-0.5B hoặc Phi-3-mini trên Orange Pi 5 / Rockchip board với llama.cpp, expose REST API.

llama.cpp, RKNN, C++, ARM NEON

3. Multi-protocol IoT Gateway

Thu thập data từ sensor I2C/SPI/UART → đẩy lên AWS IoT Core qua MQTT-TLS, có web dashboard.

C, Mosquitto, libgpiod, OpenSSL, AWS IoT

4. Linux Char Device Driver

Viết driver cho 1 sensor I2C (BME280) hoặc 1 IP FPGA, có sysfs interface. Bonus: contribute upstream.

Linux kernel module, Device Tree

5. Yocto Custom Image

Tự build Yocto image cho Raspberry Pi với app C++ của mình tích hợp sẵn, có OTA update.

Yocto, BitBake, systemd, RAUC/Mender

6. FPGA SHA-256 Accelerator

Implement SHA-256 trên Tang Nano hoặc Zybo, giao tiếp ARM qua AXI, compare hashrate vs CPU.

Verilog, Vivado, AXI4-Lite, Linux UIO
